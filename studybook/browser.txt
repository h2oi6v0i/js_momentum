// NOTE: Window
- 웹 브라우저의 전체 창을 나타내며, 웹 페이지의 전역 객체
- 웹 페이지의 크기와 위치, 브라우저의 스크롤 위치 등과 같은 정보를 제공한다.
- 다양한 메소드를 제공하여 브라우저의 창 크기 조절, 새 창 열기, 타이머 설정 등과 같은 기능을 수행할 수 있다.


// NOTE: Document 객체
- 현재 페이지의 HTML 문서를 나타내며, 웹 페이지의 구성 요소에 대한 정보를 제공한다.
- 이 객체를 통해 웹 페이지의 HTML 태그, 텍스트, 이미지 등과 같은 모든 내용에 접근할 수 있다.
- 웹 페이지에 존재하는 HTML 요소에 접근하고자 할 때는 반드시 Document 객체부터 시작해야 한다.
- 웹 페이지에 새로운 요소를 추가하거나, 기존 요소를 수정 또는 삭제하는 등의 기능을 수행할 수 있다.


// NOTE: Document.querySelector()
- 제공한 선택자 또는 선택자 뭉치와 일치하는 문서 내 첫 번째 Element를 반환한다.
- 일치하는 요소가 없으면 null을 반환한다.


// NOTE: ClassName
- 기존의 클래스들은 상관하지 않고 클래스를 교체해버린다.
- 여러 개의 클래스들 중 하나만 조작하고 싶을 때 사용할 수 없다.


// NOTE: ClassList
- element.classList.remove('클래스');
  : 클래스 제거

- element.classList.replace('클래스');
  : 클래스 교체

- element.classList.contains('클래스');
  : 존재하면 true, 존재하지 않으면 false 반환

- element.classList.toggle('클래스');
  : 존재하면 클래스 제거, 존재하지 않으면 클래스 추가
